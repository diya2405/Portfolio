## 🌐 Portfolio Website

This repository also contains my **personal portfolio website** built with:

- **React 19** + **Vite 7** – Fast frontend tooling
- **Tailwind CSS 3** – Utility-first styling with dark mode
- **Framer Motion** – Smooth animations and transitions

### Run Locally

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### Deployment

The `dist/` folder can be deployed to any static hosting provider (Netlify, Vercel, GitHub Pages, etc.).

1. Push to the `main` branch (or run the workflow manually from the Actions tab).
2. In GitHub **Settings → Pages**, set the source to **GitHub Actions**.
3. The workflow in `.github/workflows/deploy.yml` will build and publish the `dist/` folder.
